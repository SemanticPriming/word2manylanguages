# eval_inputs/replication/

Ground-truth replication norms used by [`03_evaluation/`](../../03_evaluation/) to replicate the original van Paridon & Thompson norm predictions (`evaluate_replication()` in `evaluation.py`). Contents are gitignored (see `.gitignore`) — this README stands in for the data. There's no `af` (Afrikaans) worked example here, unlike `eval_inputs/norms/`, since none of these source datasets cover Afrikaans.

## 📄 Files

- `{lang}-{author}-{year}.tsv` — one dataset, `{lang}` an ISO 639-1 code (e.g. `en`, `es`, `nl`). Tab-separated, `word` as the first column, one or more predicted-variable columns after it, column names untouched from the original source (not renamed to this project's `_mean` convention).

`03_evaluation/evaluation.py`'s `load_replication_norms(lang)` loads every file whose name starts with `{lang}` and predicts **every** non-`word` column in it — unlike `eval_inputs/norms/`, nothing here is filtered down to a curated construct list.

See `../datasets_replication.csv` for a full catalog of every file, language, and column, with each column mapped to a normalized construct name (`valence`, `arousal`, `sensory`, etc., or `other` where no clean mapping exists) alongside its original name — generated by `../build_datasets_replication.py`.
